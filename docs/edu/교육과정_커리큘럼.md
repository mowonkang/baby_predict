# 한국 교육과정 커리큘럼 참고 (2022 개정 · 누리과정)

> 딥리서치(웹) 기반 정리 — EduPath '학년별 할 일'을 **실제 국가 교육과정**에 정초하기 위함.
> 코드 반영: `backend_edu/app/grades.py`(GRADE_PLAN). 상세·정확본은 교육부 고시 원문 참조.
> 조사 기준 2026-06. 핵심 수치/단원은 출판 전 원문 재확인 권장.

## 0. 적용 시기 (2022 개정 교육과정)
- 초1·2: 2024 / 초3·4·중1·고1: 2025 / 초5·6·중2·고2: 2026 → 2027 전 학년 완료.
- 핵심역량 6: 자기관리·지식정보처리·창의적사고·심미적감성·**협력적소통**·공동체.

## 1. 유아 (누리과정, 2019 개정)
5개 영역: **신체운동·건강 / 의사소통 / 사회관계 / 예술경험 / 자연탐구** (놀이 중심).

## 2. 초등학교
- **편제**: 1·2학년 = 국어·수학 + **바른생활·슬기로운생활·즐거운생활**(통합). 3학년~ = 국어·사회/도덕·수학·과학/실과·체육·예술·**영어**.
- **영어는 초3부터 정규 교과**(1·2학년은 노출·놀이 수준).
- **1·2학년 한글 해득 강화**(입학초기 34h 확보) + 국어 시수 +34h → **기초 문해력** 집중. 실외놀이·신체활동 강화.

### 수학 4개 영역
수와 연산 / 변화와 관계 / 도형과 측정 / 자료와 가능성.

### 학년별 핵심(수학 위주, 확인된 분기점)
| 학년 | 수학 핵심 | 국어/기타 |
|------|-----------|-----------|
| 1 | 9까지·100까지 수, 한 자리 덧셈·뺄셈, 모으기/가르기 | 한글 해득(자모·받침), 그림책 |
| 2 | 세 자리 수, 덧·뺄, **곱셈구구(2학기)**, 길이·시각 | 받아쓰기·문장 읽고 쓰기 |
| 3 | **나눗셈**, **분수와 소수 도입**, 곱셈, 원·각 | 중심생각 파악(독해 시작), **영어 시작**, 과학·사회 시작 |
| 4 | 분수·소수의 덧·뺄, 큰 수, 각도, 막대·꺾은선그래프 | 문단 요약, 영어 어휘·읽기 |
| 5 | **약수·배수, 약분·통분, 분수의 덧·뺄**, 다각형 넓이 | 설명문·주장하는 글 독해 |
| 6 | **비와 비율**, **분수·소수의 나눗셈**, 비례식, 입체 부피·겉넓이 | 비문학 독해력, 중학 대비 |

> 칸아카데미 한국어는 **학년별 한국 교육과정 수학**을 무료 제공(예: ko.khanacademy.org/math/kor-6th-1).

## 3. 중학교
- 교과(군): 국어, 사회(역사)/도덕, 수학, 과학/기술·가정/정보, 체육, 예술, 영어.
- 수학 영역(초·중 공통): 수와 연산 / 변화와 관계 / 도형과 측정 / 자료와 가능성.

| 학년 | 수학 핵심 | 기타 |
|------|-----------|------|
| 중1 | 정수와 유리수, 문자와 식, **일차방정식**, 좌표평면·그래프, 기본 도형, 통계 | 문법·문학 기초, **자유학기 진로탐색** |
| 중2 | 유리수와 순환소수, 식의 계산, **일차함수·연립방정식**, 도형의 성질·닮음, 확률 | 내신 시험 대비 시작 |
| 중3 | 제곱근·실수, **이차방정식·이차함수**, 삼각비, 피타고라스, 원의 성질, 통계 | 고입·고교수학 연결 점검, 계열 방향 1차 결정 |
