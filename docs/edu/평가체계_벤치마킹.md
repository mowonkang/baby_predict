# 아동 성향·기질·능력 평가 체계 전수 조사 + 벤치마킹 반영

> 목적: "현재 기준으로는 페르소나·수준 파악에 한계가 있다"는 피드백에 따라, **공신력 있는
> 평가·발달 체계를 전수 조사**하고 각 체계의 구조를 서비스에 어떻게 차용했는지 정리한다.
> 코드 반영: `temperament.py`(기질) · `frameworks.py`(레지스트리) · `projection.py`(예상 능력치)
> · `techtree.py`(공인 등급·재능발달 단계). 전부 무과금(LLM 호출 없음).
> 원칙: **검사 자체를 시행하지 않고 '구조'만 차용**하며, 진단 아님을 항상 고지(표시광고법·의료법 리스크 회피).

## 1. 전수 조사 — 영역별 공신력 체계 지도

| **K-WPPSI-IV** (유아용 웩슬러) | 〃 | 2.5~7세용 웩슬러 | 참고(유아 문항 분화 시) |

## 2. 테크트리·예상 능력치의 권위 체계 (NEW — 이번에 반영)

### A. 재능 발달 이론 → 테크트리 단계
| 체계 | 내용 | 우리 반영 |
|---|---|---|
| **[Gagné DMGT 2.0](https://journals.sagepub.com/doi/10.1177/016235329902200209)** (영재교육 최다 인용) | **타고난 능력(gifts) → [개발 과정: 활동·투자·진전 + 촉매(개인내적/환경적/우연)] → 재능(talents)** | ✅ 티어=개발 과정 단계, 예상 능력치의 투자·촉매 시나리오 |
| **Bloom 재능발달 3단계** (『Developing Talent in Young People』 종단 연구) | 초기(놀이·흥미) → 중기(체계적 기술 습득) → 후기(전문화) | ✅ 노드 `stage`: 탐색·흥미(놀이) → 기술 습득 → 전문화(진로) |
| LTAD (장기 선수육성 모형) | 스포츠 단계별 육성 국제 표준 | 체육 트랙 참고 |

### B. 공인 등급·급수 = 실존하는 테크트리 → 노드에 부착
| 분야 | 공인 체계 | 노드 반영(`cert`) |
|---|---|---|
| 어학 | **CEFR**(유럽평의회 국제표준, A1~C2) | 노출(Pre-A1)→A1~A2→B1→B2+ |
| 수학·코딩 | 교내 경시→**HME/KMO 예선→KMO 본선**, **KOI**(정보올림피아드), COS | think1~3 |
| 과학 | 학교 과학전람회→교육청 영재원→**전국과학전람회·과학올림피아드** | sci1~3 |
| 음악 | **ABRSM**(영국 왕립음악원 1~8급)·콩쿠르 | art1~3 |
| 체육 | **국기원 품·단**(태권도), **대한수영연맹 급수** | phys1~3 |
| 인문 | **한국사능력검정**(국사편찬위), **한자급수**(한국어문회) | human1~3 |

> 등급 취득이 목적이 아니라 **객관적 이정표**(진행도 측정·동기)로 사용함을 명시.

### C. 예상 능력치 (`projection.py`)
- **프레임**: WHO·질병관리청 **소아 성장도표(백분위 트래킹)** — "현재 밴드 유지"가 기본 축.
- **시나리오 논리**: Gagné DMGT — 재능은 **개발 투자 없이는 발달하지 않는다**.
  - 보수 = 투자 적음(학교 교육만, 밴드 유지에 가까움)
  - 기본 = **추천 루트 이행**(연 +4/대상 축)
  - 낙관 = 루트 이행 + **촉매 우호**(동기·기질의 의도적 조절, 부모 지지, 잘 맞는 지도자)
- 전망 시점 = 다음 학교급 전환(초등/중등/고등 진입). 낙관/기본/보수 **3밴드 + 근거(drivers) 공개**.
- ⚠️ **정직성**: 능력 발달의 개인 예측은 과학적으로 미검증 — "보장이 아닌 참고 전망"을 응답·화면에 명시.

## 3. 반영 아키텍처 요약

```
입력(또래 대비 5단계) ─┬─ 인지 5영역(K-WISC-V 착안)  ──┐
                      ├─ 기질 3요인(CBQ 착안)  ────────┼─→ 페르소나(흥미×성향×기질)
                      ├─ 흥미(RIASEC)·사교육 경험 ──────┤
                      └─ 성취 하위스킬(또래 백분위) ────┘
                                   ↓ 삼각측량
                       8각 능력치(Gardner MI 정렬)
                                   ↓
        테크트리(Bloom·DMGT 단계 + 공인 등급 cert) → 추천 루트
                                   ↓
        예상 능력치(성장도표 + DMGT 촉매, 낙관/기본/보수 3밴드)
```

**API**: `GET /api/temperament-items` · `POST /api/temperament` · `POST /api/projection` ·
`GET /api/frameworks`(투명성 공개) + 기존 `/api/cognitive` `/api/stats` `/api/techtree` 확장.

## 4. 안전·법적 가드
- 모든 프로파일은 **참고용**이며 임상 검사(K-WISC, K-DST, CBQ 원척도)를 시행·대체하지 않음을 화면·응답·문서에 3중 고지.
- 발달 지연 의심 시 영유아 건강검진(K-DST)·전문의 상담 안내.
- 기질은 "좋고 나쁨이 없다"(조화의 적합성) — 서열화 금지.
- 예상 능력치는 미검증 참고 전망 — 효과 보장 표현 금지(공정위 표시광고 제재 사례 준수).
