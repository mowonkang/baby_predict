## 2. EduPath 현재 위치
- **유일한 통합 포지션**: 대부분 경쟁사는 *한 구간(과외/한 과목/입시)·한 기능*에 특화. EduPath는 **0세~진로 전 생애주기 + 적성 + 학년별 교육과정 + 무료/공공 우선 + 학원·리뷰**를 한 흐름으로 — "성장 동반자(Companion)" 포지션.
- **현재 보유**: 학년별 할 일·단원/무료강의 링크·자가 성취체크·약점 보완(무료/저렴+학원)·적성/진로 마인드맵·AI역량 축·유망 커리어·성취 이력·생애주기·학원 추천/리뷰·동기화.

## 4. 진화 로드맵 (우선순위)

### 1순위 — 🤖 AI 학습 튜터 (G1) ★최우선
- "이 단원 모르겠어요" → **Claude API 대화형 풀이·단계 힌트·오답 설명·글쓰기 첨삭**.
- 우리가 가장 잘 만들 수 있는 영역(LLM). Khanmigo는 유료 — **무료/저가로 제공 시 강력한 차별**.
- 단원/약점과 연결: "이 단원 막히면 → AI 튜터에게 묻기".

### 2순위 — 🎯 단원별 미니 진단(적응형) (G2)
- 자가체크(잘함/보통/부족) → **실제 3~5문항 진단**으로 정밀화, 오답을 **단원 무료강의 + AI 튜터**로 연결. IXL식 약점 정밀화.

### 3순위 — 🤝 상담/체험 신청 리드 + 리뷰 신뢰장치 (G3·G4)
- 학원 추천을 **무료 상담/체험 신청(리드)**로 연결 → 제휴 수수료(수익 직결).
- 리뷰에 **수강 인증·신고·도배 방지·랭킹**(김과외/숨고식) 추가.

### 4순위 — 🎓 진학 데이터 연결 (G5)
- 고등 path를 **대학어디가(공공) 학과·전형 데이터**와 연결, 선택과목→전형 적합도·수행평가 주제 추천(진학사식). 공공 데이터라 신뢰·무료 우선과 정합.

### 유지·강화할 해자
- **무료/공공 우선 큐레이션**(Khan 무료 vs IXL/밀크T 유료) + **전 생애주기 통합** + **적성 기반** — 경쟁사가 쉽게 못 따라오는 조합. 정직한 효과 커뮤니케이션(고가·불안 마케팅과 차별).

## 5. 한 줄 결론
경쟁사는 "검색/과외/한 과목/입시" 단일 기능 강자. EduPath의 길은 **① AI 튜터로 '실시간 학습 도움'을 채우고**, ② 자가체크를 실제 진단으로 정밀화하며, ③ 학원 추천을 거래로 완결(+리뷰 신뢰), ④ 진학 데이터로 고등 실효를 높이는 것 — 그러면서 **무료·전 생애주기·적성**이라는 통합 포지션을 해자로 유지.
