# 추천/예측 + 교육 path 엔진 상세 설계

> 상위 문서: [`기획안.md`](기획안.md)
> 본 문서는 "학생 적성 + 학령 → 커리큘럼 추천 + 장기 교육 path" 엔진 설계를 다룬다.
> 아기 트랙([`../추천엔진_설계.md`](../추천엔진_설계.md))과 동일한 구조를 교육 도메인에 적용.

---

## 1. 목표
입력된 학생 정보로 **(a) 인기 커리큘럼, (b) 적성 맞춤 리소스, (c) 단계 핵심 교과/역량**을 추천하고, 추가로 **미취학~대학 교육 path(진로 계열 로드맵)**를 생성한다. 각 산출물에 **설명(근거)**을 붙이며, 콜드스타트에서도 동작한다.

---

## 2. 입력 피처 (Feature Set)
| 피처 그룹 | 항목 | 설명 |
|-----------|------|------|
| **흥미 적성** | RIASEC 6차원 | 진로/계열 추천의 핵심 신호 |
| **학습성향** | 자기주도성·협동선호·분석성향 | 학습법·리소스 형태 가중 |
| **학령** | 만 나이(학년) | 단계 매핑의 강력한 신호 |
| **과목 선호/성취** | 선호 교과, 성취 수준 | 단계 핵심 추천 보정 |
| **환경/경제** | 거주지, 예산 | 리소스 필터 |
| **행동(축적 후)** | 클릭/수강/완료 로그 | 협업필터링·개인화 |

### 2.1 흥미 모델 — Holland RIASEC
- 6유형: **R**ealistic(현실)·**I**nvestigative(탐구)·**A**rtistic(예술)·**S**ocial(사회)·**E**nterprising(진취)·**C**onventional(관습). 각 0~1.
- 진로·계열 추천에 널리 쓰이는 검증 이론(아기 트랙의 Thomas&Chess 기질에 대응).
- 상위 1~2 유형의 조합으로 **진로 계열 path 템플릿**을 선택.

### 2.2 학습성향 (보조 차원)
- `self_direction`(자기주도) · `collaboration`(협동선호) · `analytical`(분석성향). 각 0~1.
- 추천 리소스 형태(자기주도형 인강 vs 관리형 학원, 토론/프로젝트 vs 문제풀이)와 학습법 설명에 사용.

### 2.3 적성 진단 설문 (예시 설계)
- 흥미 12문항(유형당 2문항) + 학습성향 6문항, 리커트 1~5, 역채점 지원.
- 문항→차원 매핑 후 합산·정규화 → 흥미 벡터 + 학습성향 벡터.
- 학령에 따라 문항 셋 분기 가능(초등/중등/고등 표현 난이도).

---

## 3. 출력 (Output)

### 3.1 추천
```json
{
  "recommendations": [
    {
      "resource_id": "R201", "name": "중등 과학 탐구 실험 강좌", "area": "과학",
      "type": "aptitude_match", "score": 0.91, "cost": 120000,
      "reasons": ["탐구형(I) 적성에 잘 맞는 실험 중심 강좌",
                  "중등 단계 핵심 교과(과학) 심화"]
    }
  ]
}
```
- `type`: `national_pick`(인기 커리큘럼) / `aptitude_match`(적성 맞춤) / `stage_core`(단계 핵심).

### 3.2 교육 path
```json
{
  "recommended_track": "이공/공학 계열",
  "rationale": ["탐구형(I)·현실형(R) 흥미 우세"],
  "milestones": [
    {"stage": "초등 고학년", "focus": ["수·과학 사고력", "코딩 입문"],
     "activities": ["과학 실험 키트", "수학 사고력"], "decision": null},
    {"stage": "중등", "focus": ["수학·과학 심화"],
     "activities": ["과학 탐구대회"], "decision": "특목/영재 준비 여부 검토"},
    {"stage": "고등", "focus": ["이과 내신·수능 수학/과학"],
     "activities": ["연구·R&E 활동"], "decision": "세부 전공·정시/수시 전략"},
    {"stage": "대학", "focus": ["공학/자연계열 진학"], "activities": [], "decision": null}
  ]
}
```

## 4. 단계별 모델 전략 (점진적 고도화)
- **Stage 1 — 규칙기반 + 지식 매핑(MVP/콜드스타트)**: 학령→단계 핵심 교과, RIASEC→계열/리소스 affinity 규칙.
- **Stage 2 — 인기 랭킹**: 적성×학령 코호트별 인기 커리큘럼("국민템").
- **Stage 3 — 협업 필터링**: 수강·완료 로그 축적 후 "비슷한 적성·학령 학생이 함께 들은" 추천.
- **Stage 4 — 하이브리드 + ML 랭킹**: 콘텐츠+CF+맥락(적성·성취) 러닝-투-랭크.

---

## 5. 학령 단계 매핑 테이블 (예시)
> 운영 테이블은 교육과정·전문가 검수 + 공공데이터(NEIS 등)로 확장.

| 단계 | 만 나이(예) | 핵심 역량/과업 | 핵심 교과 영역 | 추천 활동 유형 |
|------|-------------|----------------|----------------|-----------------|
| 미취학 | 3~6 | 기초 언어·수 개념, 놀이 학습 | 한글, 기초수, 예체능 | 놀이·체험 |
| 초등 저학년 | 7~8 | 문해력·기초연산, 학습 습관 | 국어, 수학, 영어 입문 | 독서·사고력 |
| 초등 중학년 | 9~10 | 사고력 확장, 흥미 탐색 | 국·영·수, 과학/사회 | 탐구·예체능 |
| 초등 고학년 | 11~12 | 심화·선행 판단, 진로 탐색 | 국·영·수 심화 | 사고력·대회 |
| 중등 | 13~15 | 교과 심화, 고입·계열 준비 | 국·영·수·과·사 | 탐구대회·동아리 |
| 고등 | 16~18 | 내신·수능·입시 전략 | 계열별 핵심 교과 | 연구·세특 활동 |
| 대학 | 19~22 | 전공·진로 실현 | 전공/교양 | 인턴·연구 |

- **적성 보정 예시**: 같은 중등이라도 *예술형(A)↑* → 예체능 실기·포트폴리오 가중, *탐구형(I)↑* → 수·과학 심화 가중.

---

## 6. 교육 path 생성 로직 (`pathway.py`)
1. 흥미 벡터에서 **상위 RIASEC 유형** 추출.
2. 상위 유형 조합 → **진로 계열 path 템플릿** 선택:
   - I/R↑ → 이공/공학, A↑ → 예술/디자인, S↑ → 교육/사회, E↑ → 경영/창업, C↑ → 상경/행정, S+E↑ → 사회·경영.
3. **현재 단계부터 대학까지** 각 미래 단계에 대해 템플릿 + 단계 매핑을 결합해 마일스톤 생성
   (focus 영역 · 핵심 활동 · 의사결정 포인트: 계열/고교유형/입시).
4. 추천 계열·근거(우세 유형) 텍스트 생성.

---

## 7. 추천 파이프라인 (런타임)
```
입력(프로필+적성)
   → [후보 생성]  단계 규칙 + 인기 코호트 + (CF 후보)
   → [필터링]     학령/예산/중복 제외
   → [랭킹]       인기·적성친화·단계적합 가중 점수
   → [설명 생성]  점수 기여 요인 → reasons
   → [광고 병합]  별도 슬롯(AD 표기)
   → 추천 + [path 생성] 반환
```
점수: `0.40·인기도 + 0.35·적성친화도 + 0.25·단계적합도` (아기 트랙 가중치 패턴 동일).

---

## 8. 평가 (Evaluation)
- **오프라인**: precision@k, NDCG, 추천 다양성, 콜드스타트 품질.
- **온라인**: A/B — 적성 path 추천 vs 인기 추천 → CTR·수강 전환·path 만족도.
- **가드레일**: 결과 보장 표현 차단, 광고-추천 분리, 편향 모니터링, 공교육/무료 자원 포함률.

---

## 9. 콜드스타트 요약
| 대상 | 데이터 없음 시 |
|------|----------------|
| 신규 학생 | 학령 규칙 + 적성 매핑 + 코호트 인기 |
| 신규 리소스 | 콘텐츠(영역·적성 태그) 기반 매칭 |
| 신규 지역 | 전국/유사지역 인기 대체 |

## 10. 향후 확장
- 성취 데이터 연동(진단평가)으로 난이도 적응 추천.
- 종단 추적(학년 진급)에 따른 path 자동 갱신·알림.
- 진학 결과 피드백 루프로 path 템플릿 보정.
