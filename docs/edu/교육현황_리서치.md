## 3. 대입 제도 + 2028 개편

### 3.1 현행(2025학년도)
- 전체 모집 약 **34.1만 명** 중 **정시 20.4%(6.9만)**, **수시 ≈ 79.6%**.
- 지역 격차: **수도권 정시 34% / 수시 65%**, **비수도권 수시 89%**.
- 수시 전형 구성: **학생부교과 45.2%(15.6만), 학생부종합 23.7%(8.2만)**. 수도권은 학종>교과(44.1% vs 32.4%), 비수도권은 교과 압도(68.2%).

### 3.2 2028 대학입시제도 개편 (2023.12.27 교육부 확정 → 2028학년도 대입부터)
- **통합형 수능**: 국어·수학·탐구 **선택과목 폐지**, 사·과탐은 **통합사회·통합과학** 출제(전원 동일 응시), **심화수학(미적분Ⅱ·기하) 미도입**.
- **내신 9등급 → 5등급제**: 절대평가+상대평가(석차등급) 병기(사실상 상대평가). 일부 융합선택 사·과 9개 과목은 절대평가만.
- 영향(보도): 5등급제로 교과전형 변별 약화 → 수도권 대학 학종·논술 확대 관측.

- **시사점(진로 계열 path)**: 통합형 수능·5등급제 환경에서 **내신 관리 + 세특·탐구활동(R&E)**의 비중이 커짐 → path 마일스톤(중·고)에 "탐구/세특 활동", "계열 의사결정 시점"을 명시적으로 반영.

출처: [교육부 2028 대입개편 확정안](https://www.moe.go.kr/boardCnts/viewRenew.do?boardID=294&boardSeq=97551), [행복한교육(교육부)](https://happyedu.moe.go.kr/), [경향신문](https://www.khan.co.kr/article/202312271100001), [에듀진 — 2025 정시 비율](http://www.edujin.co.kr/news/articleView.html?idxno=44661), [한국경제 생글생글](https://sgsg.hankyung.com/article/2025050917841)

## 4. 진로 · 적성 프레임워크

- **커리어넷**(교육부·KRIVET, career.go.kr): 중·고생용 **진로심리검사 5종** — 직업적성검사, 직업가치관검사, 진로성숙도검사, **직업흥미검사 K형/H형**. 무료 공공 서비스.
- **Holland(RIASEC)**: 직업흥미검사의 "일반흥미유형"이 Holland 6유형(**현실 R·탐구 I·예술 A·사회 S·진취 E·관습 C**) 기반. **EduPath의 흥미 모델과 동일 이론** → 공공 검사와 호환·연계 설계 가능.
- **워크넷**(고용노동부): 청소년 직업선호도검사도 Holland 이론 기반.
- 학교 진로교육: **자유학기 진로탐색활동**, 진로교육 집중학년·학기제.

- **시사점(적성 진단 설계)**: EduPath 설문(RIASEC)을 커리어넷 직업흥미검사와 **결과 호환**되게 설계하면 신뢰·연계성↑. 단, 법·윤리상 **'진단'이 아닌 '참고'**로 한정(아래 6장).

## 6. 규제 · 윤리

- **학원법(학원의 설립·운영 및 과외교습에 관한 법률)**:
  - **교습비 게시 의무**(학습자가 보기 쉬운 곳), 모집 광고 시 필수 정보 표시.
  - 게시·등록액 **초과 징수 금지**, **허위 표시·게시 금지**. **허위·과장광고로 인한 수강 철회 시 환불** 규정.
  - 위반(교습비 미게시/허위 게시): **과태료 최대 300만 원**.
- **표시광고법 관점**: 입시 결과·성적 향상 **'보장' 표현은 기만·과장광고 소지** → 금지.
- **개인정보보호법**: 만 14세 미만 **법정대리인 동의**, 미성년 학습·진로 데이터는 민감·최소수집·암호화.

- **시사점(수익화/정책)**: EduPath의 업체(학원·인강) 입점·광고는 교습비 게시·허위광고 금지 준수 필요. 플랫폼 카피에서 **"합격/성적 보장" 류 표현 금지**, 적성검사는 참고용 명시, 미성년 데이터 동의 절차 필수. → 기획안 §12(리스크/규제)가 실제 법령과 정합함을 확인.
