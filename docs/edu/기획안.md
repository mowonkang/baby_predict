# 학생 적성 기반 교육 커리큘럼·진로 path 추천 플랫폼 기획안

> **프로젝트**: `baby_predict` — 교육 트랙(EduPath)
> **문서 버전**: v0.1 (2026-06)
> **문서 종류**: 종합 사업·제품 기획안
> 본 트랙은 아기용품 추천 플랫폼([`../기획안.md`](../기획안.md))과 **동일한 구조**를 교육 도메인에 적용한 것이다.

---

## 0. 한 줄 정의 (Elevator Pitch)

> **"우리 아이 적성만 진단하면, 지금 해야 할 커리큘럼과 미취학부터 대학까지 이어지는 교육 로드맵을 그려주는 진로·학습 길잡이."**

학부모·학생이 "지금 뭘 공부시키고, 앞으로 어떤 길로 가야 할지" 고민하지 않도록, 학생의 **흥미·적성(RIASEC) · 학습성향 · 학령**을 진단해 **현재 단계 추천 커리큘럼**과 **장기 교육 path(진로 계열 로드맵)**를 제안한다. 수익은 **교육 업체(학원·인강·출판·컨설팅)의 입점/구독료 + 광고**로 만든다.

---

## 1. 배경 & 문제 정의

### 1.1 왜 지금인가
- **세계 최고 수준의 사교육 지출**: 한국 초·중·고 사교육비 총액 **약 29.2조 원**(2024, 전년 대비 +7.7%, 4년 연속 역대 최고), 1인당 월평균 **47.4만 원**, **참여율 80.0%**. (통계청 「2024년 초중고 사교육비조사」 — 근거·출처: [`교육현황_리서치.md`](교육현황_리서치.md))
- **정보 과부하 + 단발 의사결정**: 학원 선택, 선행 여부, 계열·고교유형·입시 전략이 파편적으로 결정되고, "우리 아이 적성"과 연결되지 않는다. 게다가 **2028 대입개편**(통합형 수능·내신 5등급제)으로 의사결정 기준이 재편되는 시점.
- **에듀테크 성장**: 국내 에듀테크 시장 **약 10조 원**(2025 전망), AI 맞춤학습 보편화로 *적성 기반 큐레이션* 수요 증가. (근거: [`교육현황_리서치.md`](교육현황_리서치.md))

### 1.2 핵심 문제 (Pain Points)
| 사용자 | 핵심 페인 |
|--------|-----------|
| 학부모 | "지금 뭘, 앞으로 어떤 길로?" — 적성과 무관한 남들 따라가기식 사교육 |
| 학생(중·고·대) | 흥미·강점과 진로/계열의 연결이 막연함 |
| 교육 업체 | 적성·학령이 맞는 고관여 학부모/학생에게 정밀 도달할 채널 부족 |

### 1.3 기존 대안의 한계
- **입시 카페/유튜브**: 정보는 많으나 일반론·광고 혼재, 우리 아이 적성과 무관.
- **학원 상담**: 해당 학원 상품 위주의 편향.
- **진로적성검사(학교)**: 단발성·결과 활용 부족, *장기 로드맵·커리큘럼 추천*과 단절.

> **포지셔닝**: "진로적성검사"와 "사교육 마켓" 사이의 빈 공간 — **적성 기반 장기 교육 path 큐레이션**.

## 2. 타깃 사용자 (페르소나)

### A — 미취학/초등 저학년 학부모 "은지" (자녀 만 6세)
- 한글·영어·수학 시작 시점, 사고력/예체능 방향 고민. 니즈: 적성에 맞는 **첫 방향**.

### B — 초등 고학년 학부모 "성호" (자녀 만 11세)
- 진로 탐색 시작, 선행·심화 판단. 니즈: 강점 기반 **중장기 로드맵**.

### C — 중등 학부모/학생 "민서" (중2)
- 고교 유형(일반/특목/자사)·계열 선택 임박. 니즈: 적성-계열 **의사결정 지원**.

### D — 고등 학생/학부모 "지훈" (고1)
- 문이과/세부 전공·입시 전략. 니즈: 적성 기반 **입시·진로 전략**.

> MVP 1차 타깃은 **B/C(초등 고학년~중등)** — 진로 의사결정 수요가 크고 객단가가 높다.

## 3. 핵심 가치 제안 (Value Proposition)

### 3.1 사용자(B2C)
- **방향성**: 남 따라가기 대신 *우리 아이 적성*에 맞춘 커리큘럼.
- **장기 로드맵**: 미취학~대학을 잇는 단계별 마일스톤·의사결정 가이드.
- **설명 가능**: "왜 이 과목/길인지" 근거 제시 → 신뢰.

### 3.2 업체(B2B)
- **정밀 타깃팅**: "탐구형(I) 적성 중2"처럼 적성·학령 코호트 단위 도달.
- **고전환 채널**: 진학 의사결정 직전의 고관여 학부모/학생.
- **데이터 인사이트**: 코호트별 수요/반응 리포트.

## 4. 제품 정의 & 핵심 기능

### 4.1 사용자(B2C)
1. **학생 프로필 & 적성 진단**
   - 입력: 학령(만 나이/학년), 거주지, 과목 선호/성취, 예산.
   - 적성 진단 설문(흥미 RIASEC + 학습성향) → 적성 프로필. (상세: [`추천엔진_설계.md`](추천엔진_설계.md))
2. **맞춤 커리큘럼 추천 피드**
   - ① **인기 커리큘럼(국민템)** + ② **적성 맞춤** + ③ **단계 핵심 교과/역량**. 각 추천에 **추천 이유** 표시.
3. **교육 path 타임라인 (차별 기능)**
   - 현재 단계 → 대학까지 **단계별 마일스톤**(집중 영역·핵심 활동·계열/고교/입시 의사결정 포인트)과 **추천 진로 계열**.
4. **학습 리소스 비교·리뷰·위시리스트**
   - 인강/교재/학원/대외활동 비교, 적성 태그 기반 리뷰 필터.

### 4.2 업체(B2B)
- **입점/온보딩**: 리소스 등록, 적성·교과·학령 태깅.
- **광고 캠페인**: 추천 상위 노출·배너·스폰서드(명확한 "광고/AD" 표기).
- **성과 대시보드**: 노출·클릭·전환, 코호트 반응.

### 4.3 신뢰 정책 (중요)
- 광고/스폰서 추천 **항상 명시**, 추천 점수와 광고 슬롯 **분리**.
- **입시 결과 보장 표현 금지**, 공교육·무료 자원도 함께 큐레이션(균형).

## 5. 비즈니스 모델 (수익화)

| 수익원 | 설명 | 과금 |
|--------|------|------|
| **입점/구독료 (B2B SaaS)** | 학원·인강·출판·컨설팅 입점·노출·대시보드 | 티어별 월/연 |
| **광고** | 추천 상위 노출·배너·스폰서드 | CPC/CPM/정액 |
| **제휴 커미션 (옵션)** | 수강/구매 발생 시 | 매출 연동 % |
| **유료 리포트 (확장)** | 심층 적성·진로·로드맵 컨설팅 리포트 | 건별/구독 |

### 입점 티어 (예시)
| 티어 | 월 구독료(예시) | 제공 |
|------|------------------|------|
| Basic | 무료/저가 | 리소스 등록, 기본 노출, 기본 통계 |
| Pro | 중가 | 추천 가중 입찰, 캠페인, 상세 리포트 |
| Premium | 고가 | 코호트 타깃 광고, 전담 지원, API |

> 단가는 파일럿 검증 후 확정(예시).

### 수익 흐름 (텍스트 다이어그램)
```
   (무료)        ┌────────────────────────────────────────┐
 학부모/학생 ───▶│        플랫폼 (EduPath)                  │◀─── 교육 업체
   진단/추천 ◀───│  적성 진단 · 커리큘럼 추천 · 교육 path     │   입점료/구독료 💰
   수강/구매     │  추천 엔진 + path 엔진 + 광고 엔진          │   광고비 💰
        │        └────────────────────────────────────────┘
        ▼                         │
   수강 전환 ─▶ 제휴 커미션 💰      ▼  코호트 인사이트·유료 리포트 💰
```
> **플라이휠**: 적성·성취·진로 데이터↑ → 추천·로드맵 정확도↑ → 사용자/전환↑ → 업체 유입↑ → 콘텐츠↑.

## 6. 추천/예측 + path 엔진 개요
> 상세: [`추천엔진_설계.md`](추천엔진_설계.md)
- **입력**: 적성(RIASEC 흥미 + 학습성향), 학령, 과목 선호, 예산, (축적 후) 학습·수강 로그.
- **출력**: ① 현재 단계 커리큘럼 추천(+이유) ② 미취학~대학 교육 path(진로 계열 로드맵).
- **단계별 고도화**: 규칙기반+지식매핑(콜드스타트) → 인기 랭킹 → 협업필터링 → 하이브리드 ML.

## 7. 데이터 전략
- **자체 수집(1st-party)**: 적성 진단, 학습·수강·클릭 로그, 성취/피드백 → 핵심 자산.
- **외부 후보**: NEIS(학교·학사), 학원/도서, 사교육비·진학 통계, Naver 트렌드. 상세 [`데이터소스.md`](데이터소스.md).
- **콜드스타트**: 적성×학령 규칙 매핑 + 인기 커리큘럼으로 첫 추천 → 로그 축적 후 개인화.

## 8. 기술 아키텍처 (하이레벨)
```
[웹/앱 클라이언트]
      │
[API 게이트웨이 / 백엔드]
   ├─ 학생/프로필 서비스
   ├─ 추천 서비스 ──▶ [추천 엔진] ◀── 피처 스토어
   ├─ path 서비스 ──▶ [교육 path 엔진]
   ├─ 광고/입찰 서비스

   └─ 업체 포털(B2B)
      │

```
- 권장: 프런트 Next.js + 모바일(RN/Flutter), 백엔드 Python(FastAPI, ML 통합 유리), 추천 초기 규칙엔진→ML.

## 9. 로드맵 / MVP
| 단계 | 기간 | 핵심 | 추천 방식 | 수익화 |
|------|------|------|-----------|--------|
| **Phase 0 — MVP** | 0~3개월 | 적성 진단 + 커리큘럼 추천 + 교육 path(웹) | 규칙기반 + 인기 | 수동 입점, 제휴 |
| **Phase 1** | 3~6개월 | 앱, 리뷰/비교, 업체 셀프서브 대시보드 | + 협업 필터링 | 구독 + 기본 광고 |
| **Phase 2** | 6~12개월 | ML 예측, 광고 입찰, 유료 리포트 | 하이브리드 ML | 광고/리포트 고도화 |

> **MVP 가설**: "적성 기반 path 추천이 단발 인기 추천보다 만족·전환이 높은가?"

## 10. 사용자 플로우
### B2C
```
가입 → 학생 프로필 → 적성 진단 설문
  → [추천 피드: 인기 + 적성맞춤 + 단계핵심] + [교육 path 타임라인]
  → 리소스 상세/비교/리뷰 → 수강/구매(제휴)
  → (학년 진급 알림) → 다음 단계 추천·로드맵 갱신
```
### B2B
```
업체 가입 → 리소스 등록 + 적성·교과·학령 태깅
  → 캠페인(타깃 코호트) → 노출/입찰 → 성과 대시보드
```

## 13. 경쟁 우위 / 해자 (Moat)
1. **적성–성취–진로 종단 데이터**: 학년이 올라갈수록 정확도가 복리로 향상.
2. **장기 path 락인**: 학생이 자라며 계속 로드맵·추천을 갱신받는 구조(생애주기).
3. **설명 가능 + 신뢰 정책**: 근거 제시 + 광고 분리 + 균형 큐레이션 → 리텐션.
4. **양면 네트워크 효과**: 학부모/학생↑ → 업체↑ → 콘텐츠/혜택↑.

## 부록 A. 용어
- **RIASEC**: Holland 직업흥미 6유형(현실형 R·탐구형 I·예술형 A·사회형 S·진취형 E·관습형 C).
- **교육 path**: 현재 학령부터 대학까지 적성에 맞춰 제안하는 단계별 학습·진로 로드맵.
- **국민(인기) 커리큘럼**: 해당 학령·적성 코호트에서 많이 선택되는 표준 커리큘럼.
