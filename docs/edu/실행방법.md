# EduPath 실행 방법 (비개발자용)

EduPath 앱을 내 컴퓨터에서 켜고 사용하는 방법입니다. 개발 지식이 없어도 따라 할 수 있습니다.

---

## 1. 준비물 — Python 설치 (처음 1회만)

이 앱은 Python으로 동작합니다. 설치돼 있지 않다면:

1. https://www.python.org/downloads/ 접속 → **Download Python 3.x** 클릭해 설치 파일 실행
2. 설치 첫 화면에서 **"Add Python to PATH"(또는 "Add python.exe to PATH")에 체크** ✅ — 매우 중요
3. **Install Now** 클릭 → 완료

> 이미 Python이 있으면 이 단계는 건너뛰어도 됩니다.

---

## 2. 앱 켜기 — 한 번에 실행

- **Windows**: 프로젝트 폴더의 **`start.bat`** 파일을 **더블클릭**하세요.
- **Mac / Linux**: 터미널에서 `bash start.sh` 실행.

그러면 자동으로:
1. 필요한 부품을 설치하고 (처음 1회만 1~2분 소요)
2. 서버가 켜지며
3. **잠시 후 브라우저에 앱이 자동으로 열립니다** (주소: http://localhost:8000)

> 검은 명령창이 함께 뜹니다. **이 창은 끄지 마세요** — 닫으면 앱도 꺼집니다. 다 쓰면 이 창을 닫아 종료합니다.

---

## 3. 앱 사용법 (3단계)

1. **나이 선택** — 우리 아이 단계(미취학~고등) 또는 만 나이를 입력하고, (선택) 월 예산 입력 → **적성 진단 시작**
2. **적성 진단** — 문항마다 1(전혀 아니다)~5(매우 그렇다)를 누릅니다. 다 안 채워도 결과는 나옵니다(채울수록 정확).
3. **결과 보기** — 아래 4가지가 한 화면에 나옵니다:
   - 🧭 **우리 아이 적성** (흥미 6유형 + 권장 학습 방식)
   - 🗺️ **교육 로드맵** (현재 단계 → 대학, 단계별 의사결정 시점)
   - 📚 **추천 커리큘럼** (적성·인기·단계 기반, 추천 이유 포함)
   - 🎒 **고교 과목 추천** (2022 개정: 공통/일반선택/진로선택/융합선택)

"처음부터 다시"로 다른 조건을 다시 볼 수 있습니다.

---

## 4. 스마트폰에서 앱처럼 쓰기 (선택)

- 앱은 **모바일 화면에 맞게** 만들어져 있어, 휴대폰 브라우저에서도 잘 보입니다.
- 휴대폰에서 쓰려면 컴퓨터와 **같은 와이파이**에 연결한 뒤, `start.bat`의 마지막 줄 `--host 127.0.0.1`을 `--host 0.0.0.0`으로 바꾸고, 휴대폰 브라우저에서 `http://(컴퓨터IP):8000` 으로 접속하세요. (컴퓨터 IP는 명령창에 `ipconfig` 입력해 확인)
- 브라우저 메뉴의 **"홈 화면에 추가"**를 누르면 아이콘이 생겨 **앱처럼** 실행됩니다(PWA).

> 정식 앱스토어 앱(네이티브)은 아직 아니며, 계획은 [`앱_디자인.md`](앱_디자인.md)에 정리돼 있습니다.

## 5. 자주 묻는 문제

| 증상 | 해결 |
|------|------|
| **`start.bat`을 더블클릭하면 창이 바로 꺼짐** | 대부분 **Python 미설치** 또는 **PATH 미체크**가 원인입니다. 위 1번대로 설치하세요. 원인을 눈으로 확인하려면 **명령프롬프트(cmd)를 열고** 프로젝트 폴더로 이동한 뒤 `start.bat`을 입력해 실행하면 오류 메시지가 창에 남습니다. (GitHub **ZIP 다운로드** 버전에서 줄바꿈 문제로 꺼지던 현상은 수정되었습니다 — 최신 파일을 받으세요.) |
| "Python이 설치되어 있지 않습니다" | 위 1번대로 설치하고 **PATH 체크** 확인 후 재실행 |
| 브라우저에 "연결할 수 없음" | 명령창에 `Application startup complete`가 뜰 때까지 5~10초 기다린 뒤 새로고침 |
| `8000 포트가 사용 중` | 다른 프로그램이 8000을 쓰는 중. `start.bat`의 `8000`을 `8001`로 바꾸고 `localhost:8001` 접속 |
| 패키지 설치 실패 | 인터넷 연결 확인 후 재실행 |
