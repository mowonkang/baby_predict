# 외부 데이터 소스 정리 (교육 트랙)

> 상위 문서: [`기획안.md`](기획안.md)
> 추천·path 엔진의 콜드스타트·트렌드·학령 콘텐츠를 보강할 외부 데이터 후보. (실연동 가용성은 구현 시 검증)

---

## 1. 학교 / 학사 / 학원

| 소스 | 활용 | 비고 |
|------|------|------|
| **NEIS 학교 검색**(kidshub) | 지역 학교 정보·코호트 | MCP: `kidshub-neis_school_search` |
| **NEIS 학사일정**(kidshub) | 학사/시험 일정 → path 마일스톤 타이밍 | MCP: `kidshub-neis_school_schedule` |
| **학원 검색**(kidshub) | 지역 학원 리소스 카탈로그 | MCP: `kidshub-search_academies` |
| **유치원/어린이집 비교**(kidshub) | 미취학 단계 인프라 | MCP: `kidshub-compare_kindergartens` |

## 2. 학습 콘텐츠 / 트렌드

| 소스 | 활용 | 비고 |
|------|------|------|
| **연령별 인기 도서**(kidshub) | 학령별 독서 커리큘럼 근거 | 국립중앙도서관 정보나루, MCP: `kidshub-get_popular_books_by_age` |
| **Naver 데이터랩**(검색/쇼핑) | 교육 키워드·교재 트렌드 | MCP: `NaverSearch-datalab_*` |
| **Naver 검색**(블로그/카페/지식인) | 입시·학습 페인포인트 정성 데이터 | MCP: `NaverSearch-search_*` |

## 3. 시장 / 진학 통계

| 소스 | 활용 |
|------|------|
| 통계청 **사교육비조사** | 시장 규모, 과목·학령별 지출 |
| 교육부/한국교육개발원 진학·진로 통계 | 계열·진학 흐름, path 템플릿 보정 |
| 대학알리미·대입정보포털 | 전공·입시 전형 데이터 |

---

## 4. 연동 시 유의사항
- **약관/라이선스**: 각 API·기관 데이터의 상업적 사용 가능 여부 확인.
- **개인정보**: 미성년 학습데이터는 집계·익명 위주, 법정대리인 동의 전제.
- **갱신 주기**: 입시·트렌드는 학사일정 연동 정기 갱신.
- **가용성 메모**: 작성 시점 환경에서 MCP 승인 채널 이슈로 라이브 호출 미수행. 구현 단계에서 실제 응답 스키마를 검증·매핑할 것.
