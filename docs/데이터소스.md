# 외부 데이터 소스 정리

> 상위 문서: [`docs/기획안.md`](기획안.md)
> 추천 엔진의 콜드스타트·트렌드·발달 콘텐츠를 보강할 외부 데이터 후보. (실연동 가용성은 구현 시 검증 필요)

---

## 1. 트렌드 / 국민템 근거

| 소스 | 활용 | 비고 |
|------|------|------|
| **Naver 데이터랩 쇼핑인사이트** | 연령/성별/기기별 육아용품 카테고리·키워드 트렌드 | "국민템" 랭킹·시즌성 근거 (MCP: `NaverSearch-datalab_shopping_*`) |
| **Naver 검색**(블로그/카페/지식인) | 양육자 페인포인트·실사용 정성 데이터 | 페르소나·리뷰 보강 (MCP: `NaverSearch-search_*`) |
| 오픈마켓 베스트셀러 | 카테고리별 인기 상품 | 자체 크롤링/제휴 시 약관 확인 |

## 2. 발달 / 영유아 공공 데이터

| 소스 | 활용 | 비고 |
|------|------|------|
| **kidshub – 영유아 성장 분석** | 월령별 성장 기준 | 발달 타임라인 (MCP: `kidshub-analyze_child_growth`) |
| **kidshub – 연령별 인기 도서** | 월령/연령 콘텐츠 추천 | 국립중앙도서관 정보나루 출처 (MCP: `kidshub-get_popular_books_by_age`) |
| **kidshub – 예방접종 일정** | 월령 이벤트(병원 방문 타이밍) | 질병관리청 (MCP: `kidshub-list_vaccinations_by_age`) |
| **kidshub – 어린이집/유치원/키즈카페** | 지역 기반 양육 인프라 | 지역 코호트·확장 기능 |

## 3. 시장 / 인구 통계

| 소스 | 활용 |
|------|------|
| 통계청 출생·인구통계 | 시장 규모, 지역 코호트 추정 |
| 산업연구원/유아용품 시장 리포트 | TAM/SAM, 객단가 트렌드 |

---

## 4. 연동 시 유의사항
- **약관/라이선스**: 각 API·사이트 이용약관 및 상업적 사용 가능 여부 확인.
- **개인정보**: 외부 데이터는 집계·익명 데이터 위주로 사용.
- **갱신 주기**: 트렌드 데이터는 시즌성 강함 → 정기 배치 갱신.
- **가용성 메모**: 본 기획 작성 시점 환경에서는 MCP 승인 채널 이슈로 라이브 호출을 수행하지 못함. 구현 단계에서 실제 응답 스키마를 검증하고 매핑할 것.
